# Mill Reef

Многостраничный сайт кузовной мастерской Mill Reef.

## Запуск

Откройте `index.html` напрямую или запустите любой статический сервер в этой папке.

## Состав

- `index.html` — главная страница и SEO-разметка.
- `services.html` — услуги и этапы ремонта.
- `cases.html` — кейсы и фотогалерея.
- `about.html` — информация о мастерской.
- `reviews.html` — рейтинг и отзывы клиентов.
- `contacts.html` — адрес, карта и способы связи.
- `estimate.html` — форма предварительной оценки.
- `styles.css` — адаптивный дизайн для компьютера и телефона.
- `script.js` — меню, анимации, галерея и форма WhatsApp.
- `privacy.html` — политика обработки данных.
- `assets/` — локальные фотографии из публичной карточки Mill Reef на Яндекс Картах.

Перед публикацией проверьте домен в canonical/OG-разметке и внесите полное юридическое наименование владельца в `privacy.html`.
